# Je1ghtxyuN's Personal Website

## Overview

This is my personal portfolio website hosted on GitHub Pages. The website showcases my work, provides information about me, and includes a guestbook for visitors to leave messages.

## Website Structure

The website consists of three main sections:

1. **About Me** - A personal introduction and background information
2. **Portfolio** - A showcase of my projects and creative works
3. **Guestbook** - A space for visitors to leave comments and messages

## Features

- Responsive design that works on different devices
- Simple and clean interface
- Easy navigation between sections
- Interactive guestbook functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript (for interactive elements)
- GitHub Pages (for hosting)

## How to Use

1. Visit the live site at https://je1ghtxyun.github.io
2. Navigate between sections using the menu
3. In the Guestbook section, leave your message and submit

This is a personal project, but feedback is welcome. If you notice any issues or have suggestions, feel free to open an issue on GitHub.
