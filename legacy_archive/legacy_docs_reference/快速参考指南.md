# Je1ghtxyuN 个人网站 - 快速参考指南

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
# 访问 http://localhost:5173
```

### 生产构建
```bash
npm run build
```

### 代码检查
```bash
npm run lint
```

### 部署到 GitHub Pages
```bash
npm run deploy
```

## 📁 核心目录结构

```
src/
├── components/     # React 组件
├── contexts/       # 全局状态管理
├── locales/        # 国际化文件
├── utils/          # 工具函数
├── App.jsx         # 主应用
└── main.jsx        # 应用入口
```

## 🎨 设计系统

### 主题颜色
- **主色调**: `#ff6b9e` (粉色)
- **渐变背景**: 线性渐变 `#fff0f5` → `#e6e9f0`
- **文字颜色**: `#2d3748` (主文字), `#718096` (副文字)

### CSS 变量
```css
:root {
  --primary: #ff6b9e;
  --primary-hover: #ff4785;
  --bg-gradient: linear-gradient(135deg, #fff0f5 0%, #e6e9f0 100%);
  --glass-bg: rgba(255, 255, 255, 0.75);
  --glass-border: rgba(255, 255, 255, 0.5);
}
```

## 🔧 构建配置

### Vite 配置要点
```js
// vite.config.js
export default defineConfig({
  base: '/',  // GitHub Pages 部署路径
  build: {
    rollupOptions: {
      output: {
        manualChunks: {
          'react-vendor': ['react', 'react-dom'],
          'firebase-vendor': ['firebase/app', 'firebase/firestore'],
        }
      }
    }
  }
})
```

## 🔄 状态管理

### Context API 结构
```jsx
// AppContext.jsx
const AppContext = createContext();

export const AppProvider = ({ children }) => {
  const [language, setLanguage] = useState('zh');
  const [theme, setTheme] = useState('light');
  
  return (
    <AppContext.Provider value={{ language, theme, setLanguage, setTheme }}>
      {children}
    </AppContext.Provider>
  );
};
```

## 🐛 常见问题解决

### 开发服务器问题
```bash
# 端口被占用
# 修改 vite.config.js 中的 server.port

# 依赖问题
rm -rf node_modules package-lock.json
npm install
```

### 构建问题
```bash
# 检查错误信息
npm run build -- --debug

# 清理缓存
rm -rf dist
```

### 部署问题
1. 检查 `vite.config.js` 中的 `base` 配置
2. 确保 GitHub Pages 配置正确
3. 检查 Firebase 配置和环境变量
